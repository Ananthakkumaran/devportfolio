# Dev Portfolio

This is my personal portfolio website, built to showcase my experience, projects, and technical skills as a Software Engineer specializing in Android and cloud-based systems.

## Live Site

https://ananthakkumaran.github.io/devportfolio/

## Tech Stack

* Astro
* Tailwind CSS
* Vite

## Features

* Responsive design
* Project showcase
* Professional experience timeline
* Clean and modern UI

## Run Locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Deployment

Deployed using GitHub Pages.

## Update & Publish Changes

After making updates:

```bash
git add .
git commit -m "Update portfolio"
git push origin main
```

Then rebuild and deploy:

```bash
npm run build
##npx gh-pages -d dist --dotfiles --force
npm gh-pages -d dist  
```
